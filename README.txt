# Node.js Simple Server

A basic Node.js HTTP server that serves a simple HTML page.

## Description

This project creates a simple web server using Node.js's built-in `http` module. The server listens on port 3000 and responds to all requests with a basic HTML page containing a greeting message.

## Installation

1. Make sure you have Node.js installed on your system.
2. Clone or download this repository.
3. Navigate to the project directory.

## Usage

Run the server with the following command:

```
node app.js
```

The server will start and listen on port 3000. You can access it by opening a web browser and navigating to `http://localhost:3000`.

## Dependencies

- Node.js
